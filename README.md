# 2-way-Associative-Cache
Comprehensive 2-way Associative Cache with potential scaling
Current Cache Size = 1 KB
Block size = 16 Bytes
Data is currently Halfword addressed, Building modular for word, and byte addressed data

Cache Functionalities:
 Data Reads to Cpu
 Writes from Cpu 
 Write backs to RAM
 Fetch data from RAM
